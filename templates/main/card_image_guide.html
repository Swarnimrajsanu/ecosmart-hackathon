How to Add Images to Cards

This guide shows different ways to add images to Bootstrap cards in your EcoSmart website.

Basic Card with Image

Example:

Card Title

This is a basic card with an image at the top.

HTML Code:

<div class="card h-100">
    <img src="{% verbatim %}{% static 'images/example-image.jpg' %}{% endverbatim %}" class="card-img-top" alt="Example image">
    <div class="card-body">
        <h5 class="card-title">Card Title</h5>
        <p class="card-text">This is a basic card with an image at the top.</p>
    </div>
</div>

Card with Dynamic Image from Model

Example:

Model Title

This card displays an image from a Django model.

HTML Code:

<div class="card h-100">
    <img src="{% verbatim %}{{ model_instance.image.url }}{% endverbatim %}" class="card-img-top" alt="{% verbatim %}{{ model_instance.title }}{% endverbatim %}">
    <div class="card-body">
        <h5 class="card-title">{% verbatim %}{{ model_instance.title }}{% endverbatim %}</h5>
        <p class="card-text">{% verbatim %}{{ model_instance.description }}{% endverbatim %}</p>
    </div>
</div>

Card with Conditional Image

Example:

Fallback Image

This card shows an icon when no image is available.

HTML Code:

{% verbatim %}{% if model_instance.image %}
    <img src="{{ model_instance.image.url }}" class="card-img-top" alt="{{ model_instance.title }}">
{% else %}
    <div class="card-img-top bg-light text-center py-5">
        <i class="fas fa-image fa-3x text-success"></i>
    </div>
{% endif %}{% endverbatim %}

Tips for Card Images

  • Image Dimensions

    For consistent appearance, use images with the same aspect ratio (e.g., 16:9 or 4:3).

  • File Size

    Optimize images for web to keep file sizes small and improve page load times.

  • Alt Text

    Always include descriptive alt text for accessibility and SEO.

  • Responsive Images

    Bootstrap's card-img-top class makes images responsive by default.

Adding Images to Your Models

Model Definition:

from django.db import models

class YourModel(models.Model):
    title = models.CharField(max_length=200)
    description = models.TextField()
    image = models.ImageField(upload_to='your_model_images/', null=True, blank=True)
    
    def __str__(self):
        return self.title

Form with File Upload:

from django import forms
from .models import YourModel

class YourModelForm(forms.ModelForm):
    class Meta:
        model = YourModel
        fields = ['title', 'description', 'image']
        widgets = {
            'image': forms.FileInput(attrs={
                'class': 'form-control', 
                'accept': 'image/*'
            }),
        }
